# Test-GPT

Этот репозиторий содержит простую игру «Змейка», написанную на Python. Чтобы запустить игру, выполните:

```bash
python -m snake.cli
```

Управляйте змейкой с помощью стрелок на клавиатуре. Закройте окно терминала или нажмите любую клавишу после сообщения об окончании игры, чтобы выйти.
